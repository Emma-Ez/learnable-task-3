# e-Commerce Console App: A Simple Shopping Cart Experience

This repository holds the source code for a basic e-commerce console application, offering a simulated shopping experience through text-based interaction. Built entirely in JavaScript, it provides a foundation for exploring core e-commerce functionalities and practicing code development.

## **Key Features:**

- **Product Browsing:** View a list of available products with prices and IDs.
- **Shopping Cart Management:** Add, remove, and adjust quantities of desired items.
- **Cart Overview:** See detailed information about items in your cart, including individual and total prices.
- **Simulated Checkout:** Process a mock payment and clear the cart upon successful completion.
- **User Input Validation:** Ensure accurate data entry with ID verification and quantity checks.
